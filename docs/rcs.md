| Kode | Metrik RCS                                                | Yang dinilai                                                                                         |
| ---- | --------------------------------------------------------- | ---------------------------------------------------------------------------------------------------- |
| M1   | **Pre-event completeness**                                | Data sebelum event tersedia dan cukup durasinya                                                      |
| M2   | **Event-window completeness**                             | Data saat event tersedia dan mencakup gejala utama                                                   |
| M3   | **Post-event completeness**                               | Data setelah event tersedia untuk melihat recovery                                                   |
| M4   | **Wi-Fi diagnostic completeness**                         | Status koneksi Wi-Fi tersedia: connected/disconnected, SSID/BSSID/RSSI/bitrate/channel jika tersedia |
| M5   | **IP configuration completeness**                         | IP address, interface, gateway/default route tersedia                                                |
| M6   | **Routing diagnostic completeness**                       | Informasi rute jaringan tersedia                                                                     |
| M7   | **DNS resolver completeness**                             | Resolver DNS yang digunakan tersedia                                                                 |
| M8   | **Service-timing / event-specific evidence completeness** | Timing/metrik layanan sesuai event tersedia                                                          |
